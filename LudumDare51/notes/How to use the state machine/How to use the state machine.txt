1) Create a a script specific to object in the State Machines group
	Example: fsm_myObject
	a) create a states struct
//		var states = {};

	b) create DN (do nothing) function
//		static DN = function () {};

	c) create states in the struct using:
//  	states[$ "StateName"] = new State 
			( ... see step 3

2) In your object:
	a) In the create event, make a state machine object
//		state_machine = new StateMachine(self, fsm_myObject(), "InitialStateName");
	b) In the step event call
//		state_machine.step();
	c) In the destroy event do
//		delete state_machine;


3) How to create states!
	a) States take in three args: functions for when you enter, step and leave the state.
		The function looks like this:
//		function (target, state_machine) {...
	b) Remember to use target to reference any variables or methods within your object, like:
//		var xPos = target.x
	c) Use DN when you want to leave an enter/leave function blank
	d) To transition to a state:
//		state_machine.transition("StateName")
